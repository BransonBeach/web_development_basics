HyperText Markup Language

You can view the source of any webpage using the "view source" tool available in all major desktop browsers via the key combination Ctl-U.
A markup language is a modern system for annotating a document in a way that is syntactically distinguishable from the text. So, HTML is a formatting and layout language for text as well as user interface elements (input boxes, menus, etc.) in documents that contain hyperlinks. HTML uses pairs of tags to specify elements. For example, all HTML documents start with an <html> tag and end with an </html> tag. Comments start with <!-- and end with -->. They can contain other tags, which will be ignored until the --> is reached.

Cascading Style Sheets

The source code for stylesheets is also available by clicking the link in the HTML source.
Cascading Style Sheets (CSS) is a style sheet language used for describing the look and formatting of a document written in a markup language. They are called "cascading" because HTML elements inherit all style attributes from their parents. This inherited style can be overridden by local style attributes. Using styles, you can control everything from colors and font properties to layout and even some dynamic behavior.
